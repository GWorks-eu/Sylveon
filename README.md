# Sylveon
Sylveon is a Server to run Node.JS Scripts. This has been tested with Discord.js

First install the Packages

```bash
$ npm install
```

Then start the Server by using node

```bash
$ node Server.js
```

Or using the Start Scripts

Linux and Mac:
```bash
$ chmod 777 ./start.sh
$ ./start.sh
```

Windows:
```bash
$ ./start.bat
```
